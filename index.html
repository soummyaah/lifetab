About LifeTab

As the name suggests it's a platform where you can keep tabs on your life. Create to-do lists, make journal entries. This can be set to open on a new tab and used from there. It also has a google search bar. It requires google login. It is written in Python. It uses Django framework for the backend and Ajax for frontend.
